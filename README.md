# LitModDyn
Simple solvers for Navier Stokes equtions in Python
LitModDyn is a new Python code as a post-processing tool for LitMod2D models, which simulates present-day mantle flow and computes dynamic topography.

The code computes present-day mantle flow and associated dynamic topography by solving the steady-state Stokes equations in a two-dimensional Cartesian domain using finite-difference and marker-in-cell techniques. The model assumes an incompressible, viscous mantle and neglects inertial forces, which is appropriate for long-term mantle flow at very low Reynolds numbers.
